# 🎓 تطبيق التوجيه الدراسي في المغرب

> 🚀 تطبيق مكتبي حديث مبني بلغة Python يساعد التلاميذ في المغرب على اختيار الشعب الدراسية ومعرفة الآفاق المستقبلية لكل شعبة.

## 📌 نظرة عامة

هذا المشروع هو *أداة للتوجيه الدراسي* تهدف إلى تسهيل عملية اختيار الشعبة المناسبة للتلميذ.
يتميز التطبيق بواجهة حديثة وسهلة الاستخدام، حيث يمكن للمستخدم استكشاف مختلف الشعب الدراسية والتعرف على الفرص المهنية المرتبطة بها.

## ✨ المميزات

* 🎯 واجهة بسيطة وسهلة الاستخدام
* 🎨 تصميم حديث باستخدام CustomTkinter
* 📚 مجموعة متنوعة من الشعب الدراسية
* ⚡ عرض سريع للآفاق المهنية
* 💻 تطبيق خفيف وسريع

## 🧠 الشعب المتوفرة

* العلوم الرياضية (A و B)
* العلوم التجريبية
* العلوم الاقتصادية
* علوم التدبير المحاسباتي
* الآداب
* العلوم الإنسانية
* العلوم الفيزيائية
* علوم الحياة والأرض
* الفنون التطبيقية


## 🛠️ التقنيات المستعملة

* Python 🐍
* Tkinter / CustomTkinter
